# galaxyvim starter

This is the official starter template for installing **galaxyvim**.

## Requirements

- Neovim 0.11+ (recommended latest stable)
- Nerd Font (recommended)
- `git`, `rg`, `node`,`npm`

## Installation

### Linux / macOS

```bash
git clone https://github.com/galaxyvim/starter ~/.config/nvim
nvim
```

### Windows

```powershell
git clone https://github.com/galaxyvim/starter $env:LOCALAPPDATA\nvim
nvim
```
